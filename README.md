# Unturned Phantom External (2026)

Modern, safe, and beautiful external ESP/Aimbot for Unturned 3.x.

## Features
- **Mono Resolver**: Dynamic offset resolution.
- **Modern UI**: Dark-themed WinForms menu.
- **Safety**: Memory caching and smooth aim.
- **Overlay**: Transparent GDI+ rendering.

## How to Build
1. Create a **Windows Forms App (.NET Framework)** in Visual Studio.
2. Add all `.cs` files from the `external/` folder.
3. Build and Run while Unturned is open.
